# 🚀 AI Resume Analyzer

AI-powered web app that analyzes resumes, gives ATS score, and provides personalized career feedback.

## 🔥 Features

* 📄 Upload Resume (PDF)
* 🤖 AI Resume Analysis (Groq API)
* 📊 ATS Score
* 🧠 Skill Breakdown (Strong / Missing)
* 💬 Human-like Suggestions
* 🎯 Job Fit Analysis

## 🛠 Tech Stack

* Frontend: React + TailwindCSS
* Backend: Node.js + Express
* AI: Groq (LLaMA 3)

## ⚙️ Setup

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm run dev

## 🔐 Environment Variables

Create `.env` in server:

GROQ_API_KEY=your_key_here

## 🚀 Deployment

* Frontend: Vercel
* Backend: Render
